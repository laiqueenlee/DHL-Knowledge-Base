# Turn Off Automatic Replies

`TurnOffAutomaticRepliesConnections`

Deactivates Gmail vacation/out-of-office automatic replies.

**Package:** `UiPath.GSuite.Activities`
**Category:** Gmail
**Connector:** `uipath-google-gmail`

## Properties

### Configuration

| Name | Display Name | Type | Default | Description |
|------|-------------|------|---------|-------------|
| `ConnectionId` | Connection ID | `InArgument<string>` | | The Google connection to use. |

## Notes

- No input properties are required. This activity simply disables the vacation auto-reply setting.
